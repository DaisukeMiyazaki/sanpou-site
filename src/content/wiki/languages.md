Sanpou supports three voice recognition languages: **English**, **Japanese**, and **Dutch**. Set your primary language in Settings — voice memos and dictation use this.

## Secondary Language

Pick a secondary language too, and you can switch between the two without going back to Settings. Handy if you live between two languages or talk to yourself in one and to others in another.

## Dutch Support

Dutch was added in 2.0.0, including a dedicated lock screen recording widget so Dutch speakers don't need to change settings each time they record.

## OCR Follows Your Language Settings

[Image OCR](/sanpou-site/wiki/image-ocr/) supports the same three languages. The OCR language picker only shows the languages you've actually enabled — primary plus the optional secondary — so single-language users don't see a toggle they'd never use, and Dutch users now see Dutch as a real option instead of being silently dropped to English.
