Weather is a subtle cue that brings a memory back. "That idea I had on the rainy walk home." From this version on, every new memo automatically captures the current weather using Apple's WeatherKit.

A small weather icon appears next to each memo in the list, and weather conditions are shown on the map detail card too.

## Filter by Weather

On the map view, you can filter memos by weather category — show only the sunny-day walks, or only the rainy ones. Useful for noticing patterns you didn't know existed.

Note: weather data is recorded for new memos going forward. Memos created before this feature was added won't have weather info.
