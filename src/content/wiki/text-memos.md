Not every moment calls for voice. From the memo list, tap the text input button and the keyboard pops up instantly. The same metadata — date, location, step count — is captured automatically, so a typed memo lives alongside your spoken ones with no second-class treatment.

## Lock Screen Text Entry

A lock screen widget lets you start a text memo without unlocking and opening the app. Tap, type, save. The widget is especially useful for moments when you want to be discreet.
